# Automatizacion-pallets-a-recibir
# ERP Report Automation (SAP GUI + Python)

## 📌 Descripción
Automatización de extracción de datos desde SAP GUI, consolidación y generación de reportes en Excel con envío automático por correo.

## ⚙️ Funcionalidades
- Automatización SAP GUI
- Procesamiento de múltiples centros (CEDIS)
- Consolidación con Pandas
- Generación de reportes Excel
- Envío automático vía Outlook

## 🧰 Tecnologías
- Python
- Pandas
- SAP GUI Scripting
- Win32com

## ⚠️ Nota
Este repositorio es una versión demostrativa anonimizada.
